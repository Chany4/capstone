Users

HTTP Methods Path Description
GET /bigTime/getUsers Retrieves all users from the database
GET /bigTime/getUser/:id Retrieves a user using the user's ID (userID)
POST /bigTime/addUser Signs up/Adds a new user to the database
PATCH/PUT /bigTime/updateUser/:id Updates/Edits/Alters user's details by ID (userID)
DELETE /bigTime/removeUser/:id Deletes a user by their ID (userID)
POST /bigTime/login Logs a user in using their email (emailAdd)

Products

HTTP Methods Path Description
GET /bigTime/getMechanics Retrieves all products from the database
GET /bigTime/getMechanic/:id Retrieves a product using the product's ID (productID)
POST /bigTime/addMechanic Add a new product to the database
PATCH /bigTime/mech/:id Update/Edits/Alters product's details by using it's ID (productID)
DELETE /bigTime/mech/:id Deletes product's details by using it's ID (productID)
